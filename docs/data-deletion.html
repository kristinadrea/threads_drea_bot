Data Deletion Instructions

Last updated: July 8, 2026

Drea Threads Bot is a private automation tool. It stores only minimal technical state, such as processed Telegram message identifiers and weekly posting state.

How to request deletion

To request deletion of data associated with this app, email kristina777@proton.me with the subject line Drea Threads Bot Data Deletion.

What will be deleted

  • Stored processed Telegram message identifiers.
  • Stored weekly posting state.
  • Any temporary files controlled by the app owner, if present.

Processing time

Deletion requests are handled as soon as reasonably possible. After deletion, the app may no longer remember which Telegram posts were previously processed.

Revoking access

You can also revoke the app's access from your Meta or Threads account settings, and you can remove the Telegram bot from any Telegram channel where it was added.
